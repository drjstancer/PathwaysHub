# CQI Report Specification

## Purpose

This document defines the minimum CQI outputs supported by the MVP.

## Preprofessional Programs

Examples:
- JPAWS
- MMRSEP
- Medical Explorations

### Required outputs

- Applications received
- Students accepted
- Students enrolled or attended
- Completion counts
- School representation
- City representation
- County representation
- Rural participation summary
- SES participation summary
- Scholarship counts
- Scholarship amounts (optional)

## Preadmission Programs

Examples:
- PAWS
- PEN
- Bryant Scholars
- Springfield Scholars

### Required outputs

- Applications received
- Students accepted
- Active students
- Graduates
- Average GPA
- Average science GPA
- Average MCAT
- Medical school matriculation counts
- Medical school destinations

## Match Outcomes

### Required outputs

- Match year summary
- Specialty distribution
- Primary care percentage
- Missouri match percentage
- Institution listing

## Executive Dashboard Metrics

- Total students served
- Active participants
- Program completers
- Medical school matriculants
- Residency matches
- Missouri matches
- Primary care matches

## Future Metrics

- Step 1 pass rates
- Block failures
- Clerkship failures
- Residency completion
- Workforce location
- Rural workforce placement
